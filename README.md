After npm install if you want the start applications
For team-task-management-hub-api go to folder give `npm start` command it will run by the port on browser(http://localhost:9999/taskListSummary)
For Task-Management-App go to give `npm run dev` command it will run by the port on browser (http://localhost:5173/)
For Task-Mangement-APP can add, edit and delete opperation on the tasks
